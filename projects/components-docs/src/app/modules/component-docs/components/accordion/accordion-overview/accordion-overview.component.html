The cui-accordion component is an accessible accordion similar to the Angular Material expansion panel. It differs in that the button to expand and collapse the accordion body is limited to just a single left justified button inside of the header panel instead of the entire header panel itself being the button. This allows us to place other actionable items (buttons, links, etc.) inside of the accordion header panel without creating any accessibility issues, because screen readers cannot correctly interpret nested buttons.

In other words, if we put a button inside of the header of the Angular Material expansion panel, then we create a situation where there is a button inside of another button, which confuses assistive technology. The CoreUI accordion provides an accessible alternative to this pattern.
