The CoreUI accordion comes with test harnesses for the Accordion and AccordionItem component classes.

Importing Test Harnesses

Test Harness Filters

AccordionHarness API - extends Angular's ComponentHarness

Name Description
async getAccordionItems(filter: AccordionItemHarnessFilters = {{ '{}' }}): Promise<AccordionItemHarness[]> Gets all accordion items which are part of the accordion.
async isMulti(): Promise<boolean> Whether the accordion allows multiple expanded accordion items simultaneously.
static with(options: AccordionHarnessFilters = {{ '{}' }}): HarnessPredicate<AccordionHarness> Gets a HarnessPredicate that can be used to search for an accordion with specific attributes.

AccordionItemHarness API - extends Angular's ContentContainerComponentHarness

Name Description
async isExpanded(): Promise<boolean> Whether the accordion item is expanded.
async isDisabled(): Promise<boolean> Whether the accordion item is disabled.
async isFocused(): Promise<boolean> Whether the accordion item toggle button is focused.
async toggle(): Promise<void> Toggles the expanded state of the accordion item by clicking on the toggle button. This method will not work if the accordion item is disabled.
async focus(): Promise<void> Focuses the toggle button.
async blur(): Promise<void> Blurs the toggle button.
async getToggleButtonPosition(): Promise<'left' | 'right'> Gets the position of the toggle button.
async getHarnessLoaderForContent(): Promise<HarnessLoader> Gets the position of the toggle button.
async getTitle(): Promise<string | null> Gets the title text of the accordion item if a mat-panel-title was used in the header.
async getDescription(): Promise<string | null> Gets the description text of the accordion item if a mat-panel-description was used in the header.
static with(options: AccordionItemHarnessFilters = {{ '{}' }}): HarnessPredicate<AccordionItemHarness> Gets a HarnessPredicate that can be used to search for an accordion item with specific attributes.
